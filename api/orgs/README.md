# Organizations

## List

This returns a list of organizations the current user is a member of.

> 列出当前登录用户的组织

| Method | URL     |
| :----- | :------ |
| `GET`  | `/orgs` |

## Find

This returns an individual organization.

> 返回某个组织的信息

| Method | URL                                      |
| :----- | :--------------------------------------- |
| `GET`  | `/org/{git_type}/{organization.name}` |
